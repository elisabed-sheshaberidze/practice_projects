# Number Guessing Game

## How to Run
File can be run directly with: python number_guess_game.py

## How to Play
The Number Guessing Game allows the player to choose a difficulty level.
Based on the difficulty, you must guess a number within a chosen range
with limited tries and an odd/even hint!

- Reach 100 points to win!
- Drop below -20 points and you lose!
- Type "quit" at any time to exit the game.

## Difficulty Levels
There are 3 difficulty levels: Easy, Medium, and Hard.

## Scoring
| Difficulty | Range | Correct Guess | Wrong Guess |
|------------|-------|---------------|-------------|
| Easy       | 1-30  | +5 points     | -2 points   |
| Medium     | 1-50  | +10 points    | -3 points   |
| Hard       | 1-100 | +20 points    | -8 points   |

Each round gives you 5 tries to guess the number.
